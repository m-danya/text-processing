1. Пароль

на каждое из следующих условий написаны lookaheadы:
	- верно, что есть хотя бы одна заглавная буква
	- верно, что есть хотя бы одна строчная буква
	- верно, что есть хотя бы одна цифра
	- неверно, что есть используется только один спецсимвол
	- неверно, что есть два одинаковых символа подряд

2. Цвета

На каждый вариант написан свой регексп, дальше они комбиринруются через nc groups с помощью питоновских форматных строк

3. Выражение

Пришлось повозиться с
    - приоритетами, чтобы variable определялись в самом конце, если нет такой функции или константы
    - \b, чтобы pipipi была переменной, а не тремя константами

4. Даты

Даты написаны прямо по заданию, никаких хитростей нет.
